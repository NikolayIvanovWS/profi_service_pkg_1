# profi_serv_1

### Installation
```
cd ~/catkin_ws/src/
git clone https://github.com/voltbro/profi_serv_pkg_1.git
```
### Compilation
```
cd ~/catkin_ws/
catkin_make --pkg profi_serv_pkg_1
```
### Usage

Just run configure.launch
```
roslaunch profi_serv_pkg_1 start_configure_1.launch
```

### Required libraries

```
time
random
tqdm
```
